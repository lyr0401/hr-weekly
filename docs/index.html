数据安全与员工隐私
《个人信息保护合规审计管理办法》落地，HR系统须建立"可核验"合规档案
《个人信息保护合规审计管理办法》自2025年5月1日起施行，推动企业合规工作从"自我声明合规"转向"可核验合规"，要求企业为后续审计建立"可审计四件套"：处理活动台账、个人信息保护影响评估材料留存、第三方处理者管理记录、用户权利响应工单与日志。对HR部门而言，员工数据收集、招聘候选人信息留存、第三方招聘平台的数据共享协议，均须纳入可审计的证据链管理，处理100万人以上个人信息的企业还须指定个人信息保护负责人，并于2025年7月起通过"个人信息保护业务系统"完成线上信息报送。
跨国企业员工数据出境有条件豁免，但"必要性"边界须严格把握
根据《促进和规范数据跨境流动规定》，"按照依法制定的劳动规章制度和依法签订的集体合同实施跨境人力资源管理，确需向境外提供员工个人信息"的情形，可免予申报数据出境安全评估、订立标准合同或通过认证；但相关处理须遵守必要、目的明确、最小化原则，仅限于与人力资源管理目的直接相关的信息。监管部门明确指出，身份证、护照、银行账户等信息是否属于"确需"范围，须逐一对照上述原则具体判断，2025年9月上海公安机关已对某跨国公司违规向境外总部传输用户个人信息的行为作出首例行政处罚，标志着数据跨境监管进入常态执法阶段。
2026年个人信息保护专项行动延续高压，招聘类App及HR系统面临重点抽查
2025年3月，中央网信办、工业和信息化部、公安部、市场监管总局联合启动贯穿全年的个人信息保护系列专项行动，全年共有三千多款App在监管通报中"榜上有名"，执法频次与力度空前。2026年专项行动延续这一态势，重点关注告知缺陷、未经同意或强制同意、超范围收集（如位置、通讯录、短信），以及技术合规不足等问题，并要求企业在人脸识别场景（如考勤、门禁）下不得将人脸作为唯一验证方式，否则构成违规。使用招聘App、员工自助系统等HR数字工具的企业，应及时完成权限与SDK合规扫描，防范因第三方组件违规引发的连带责任。
人才市场动向 Talent
科技岗位动态（算法 / 研发 / 产品）
大模型算法岗供需极度失衡，薪资领跑科技市场
2026年1-2月，AI岗位数量同比增长约12倍，在全部新经济岗位中的占比从2025年同期的2.29%跃升至26.23%。大模型算法位居热门岗位之首，AI科学家/负责人平均月薪达137,153元，算法研究员、AIGC算法工程师等岗位薪资也普遍在7万元左右；人才紧缺度方面，高性能计算工程师供需比仅为0.15，约7个岗位才能争夺到1个合适人才。
大厂研发岗扩招提速，AI产品经理成非技术岗薪资天花板
字节跳动2026届计划招聘超5000个岗位，比去年增加20%，研发类岗位招聘数量同比增长23%，算法、前端、客户端增幅最大；京东更宣布开放3.5万个校招岗位。头部企业为抢人才不惜重金，字节跳动旗下"豆包"业务为"大模型应用架构专家"开出高达128万元年薪，平台产品经理月薪上限也触及6万元。非技术岗中，仅AI产品经理能达到3万元/月，但需具备"技术理解+业务落地"双重能力。
整体科技岗薪资上行，但"去初级化"趋势明显加剧
2026年春招新经济行业新发岗位平均月薪由2025年同期的44,124元提升至48,189元，同比增长约9.2%，整体呈现"岗位增加、薪资上升、竞争趋缓"的态势。然而招聘结构加速"去初级化"——新发岗位中要求3年以上经验者占比超七成，面向1年以内经验者的岗位缩减约20%，且34.39%的新发岗位已明确要求具备AI能力。
运营与职能岗位动态
职能岗位需求两极分化，AI技能成硬门槛
运营、HR、市场营销、财务等职能岗位受AI冲击最大，招聘需求大幅缩减——企业借助AI工具和数字化手段提效，"以前一个公司需要一堆运营、一堆HR，现在几个工具加几个人就搞定了"。在仅存的招聘需求中，精通AI与数字化转型、具备出海经验的HR享有薪资溢价，而中高端HR若无稀缺技能或实战战绩，跳槽大概率面临平薪或降薪。
财务岗"战略化"转型，FP&A与出海合规人才薪资走高
经济不确定背景下，FP&A与战略财务领导力愈发凸显，企业渴求能将数据转化为战略洞察的人才；财务数字化工具（RPA、AI）加速普及，财务流程向共享中心集中并延伸至分析与情景模拟；与此同时，出海战略催生全球税务专家需求，跨境合规与本土税务并重，外派财务岗及港股IPO相关岗位（投资者关系、具备IPO经验的CFO）缺口显著。
互联网大厂职能类薪资分化，腾讯运营月薪15-20K、"AI复合型"溢价明显
从头部大厂薪资水平来看，腾讯运营职位月薪在15-20K之间，年总包约23-37万；技术岗月薪18-24K，总包33-50万，整体薪酬仍具吸引力。但市场整体结构已明显分化——具备2-3项关联技能的复合型人才（如懂业务的数据分析师、兼顾AI工具应用的运营）薪资比单一技能人才高出35%-50%，晋升速度也快20%。
应届生与校招趋势
互联网大厂2026届校招全面扩招，AI岗位成绝对主角
字节跳动计划招聘超5000个岗位，比去年增加20%，研发类岗位同比增长23%；阿里巴巴计划招聘超7000个岗位，AI相关岗位占比超六成；腾讯则将在"AI+"领域大幅增加校招投入。大厂对AI类岗位的需求量相较于往年增长达十倍左右，岗位方向更加垂直细分，细化到大模型、量子计算、机器人等领域，而Java开发、iOS工程师等基础技术岗位的招聘需求则出现大幅下滑。
AI岗薪资"天花板"被持续刷新，顶尖人才年薪不设上限
各大厂AI研发类岗位普遍能给出月薪10万左右的offer，字节跳动在招聘一位清北算法博士实习生时，甚至开出了日薪5000元。对于应届生而言，AI相关岗位年薪总包甚至会高出其他类岗位50%，针对顶尖AI人才，各大厂私下定下"不惜代价、上不封顶"原则。
应届生就业意向向AI靠拢，求职者从关注薪酬转向看重技术成长
一位2026届毕业生表示，他的实习会优先考虑和AI结合的业务内容，"尽量向AI靠拢"；诸多AI相关内容也让更多学生对职业规划产生新的思考。求职者在选择公司和岗位时，不再仅考虑薪资和业务发展前景，还会考虑技术负责人的专业能力、团队的技术突破能力以及个人技术精进等多方面因素，甚至更倾向直接与技术负责人沟通而非HR。
高端人才与猎头市场
AI大模型争夺战引发总监级技术人才大规模跨厂流动
大模型赛道的人才争夺已蔓延至总监及以上层级：字节连续两次向大模型方向技术员工授予期权激励，L4级算法总监综合薪酬可达四五百万；某竞对大厂随即开出"连升2级+薪水翻倍"的条件从字节反向挖人，部分资深算法研究员薪酬被推至千万级。
猎头渠道迎来结构性复苏，AI/互联网高端岗位需求同比暴增逾七成
2026年春节后首周，猎聘平台企业通过猎头发布的高端职位同比增长34.17%，其中AI/互联网/IT行业新发职位同比增速高达72.73%，成为增速最快的赛道之一；猎头平均推荐时长也从2025年的24.5小时大幅压缩至10.6小时，反映出AI匹配技术的介入与高端人才流动活跃度的双重提速。
互联网高管薪酬包加速分化：AI复合型人才溢价显著，传统职能总监承压
薪酬数据平台显示，互联网行业技术岗薪资领跑全行业，一线城市AI工程师薪酬涨幅显著，大模型算法岗年薪已普遍破百万，北上深杭形成高薪地效应；与此同时，互联网行业整体薪酬增速放缓、内部分化加剧，具备AI能力的业务型总监受到追捧，而传统职能方向高管的议价空间持续收窄。

百度于2025年底启动年终裁员，涉及多个业务线，各部门裁员比例不等，移动生态事业群组（MEG）受影响尤为广泛，旗下百度App、百度网盘、百度文库等业务均涉及人员调整，与AI及云计算相关的岗位基本得以保留。百度员工总数自2021年已从4.55万人持续缩减至约3.59万人，近四年累计减员逾万人，此次裁员后公司已基本全面暂停招聘并锁定HC，等待人才盘点完成后再开放。背后原因在于搜索广告主业持续承压，而All in AI的转型尚未形成有效的收入替代。
阿里巴巴·裁员
